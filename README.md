# 前言

小兔鲜项目适合已经熟练使用vue2，准备学习vue3的小伙伴作为练习的中型电商项目

## 技术栈

vue3 + vite + vue-router + pinia + ES6/7 + vueUse + sass + element-plus

## 项目设置

```sh
git clone https://gitee.com/xxmliu/vue-rabbit.git
cd vue-rabbit
npm install
```

### 项目运行

```sh
npm run dev
访问 http://localhost:5173
```

### 编译打包

```sh
npm run build
```
